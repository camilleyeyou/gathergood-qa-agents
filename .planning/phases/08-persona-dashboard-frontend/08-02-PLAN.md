       - One `<td>` per flow: the heatmap cell

   - Apply `frictionClass(score)` from utils.ts for background color (green 1-3 / yellow 4-6 / red 7-10)
   - Display friction_score as large number centered in cell

   - onClick: `setSelectedCell({ persona, flow })` — this triggers the Sheet to open

   - Cell sizing: min-w-[100px] h-[80px] for desktop, responsive padding
   - cursor-pointer and hover:opacity-80 transition for interactivity feedback

   - Pass the selected PersonaResult to DetailPanel (created in Task 2)
   - Per D-08: Panel stays open when clicking different cells — selectedCell state change swaps content without closing the Sheet

   - Add page header: "Persona Dashboard" as h1, brief description text
