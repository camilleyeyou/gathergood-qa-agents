**HTML structure:**

- A heading h1: "Digital Literacy Persona Report"
- Subtitle p: "Run: {{ run_id }}"
- An HTML table with:
  - Header row: empty cell, then one th per flow ("Registration", "Browsing", "Checkout")
  - One tr per persona in `personas` list (tech_savvy, casual, low_literacy, non_native, impatient)
  - First td in each row: persona display name (replace underscores with spaces, title case) + literacy level badge "(Level {{ results[(persona, flow)].literacy_level }})"
  - Each cell td: friction score number, background color from CSS class
  - Cell also shows task_completed status as checkmark or X icon (plain text unicode)

**CSS classes (embedded in style block):**
- `.friction-low` — background: #d4edda (green), color: #155724
- `.friction-mid` — background: #fff3cd (yellow), color: #856404
- `.friction-high` — background: #f8d7da (red), color: #721c24
- Table: border-collapse, 100% width, text-align center
- Cells: padding 12px, font-size 1.2em for scores, border 1px solid #dee2e6
- Persona name cells: text-align left, font-weight bold

**Heatmap class logic (Jinja2):**
Use a Jinja2 macro or inline conditional:
```
{% set score = results.get((persona, flow), {}).get('friction_score', 0) %}
{% if score <= 3 %}friction-low{% elif score <= 6 %}friction-mid{% else %}friction-high{% endif %}
```

**Detail section below the matrix:**
- For each persona/flow combination with confusion_points, render an expandable details/summary block:
  - Summary: "{{ persona }} — {{ flow }}: {{ confusion_points|length }} confusion points"
  - Inside: ordered list of confusion points with step number, description, severity badge
  - If screenshot_path is set, render an img tag with relative src

**Footer:**
- "Generated by GatherGood Persona Agent Suite"
- Timestamp: "{{ generated_at }}"
