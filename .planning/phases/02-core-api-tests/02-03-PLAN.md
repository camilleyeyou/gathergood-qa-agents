**TORG-01: test_create_org_assigns_owner_role(auth_client, teardown_registry)**
- POST `/organizations/` with `{"name": org_name(), "description": "Test org"}`
- assert_status(resp, 201, "POST /organizations/")
- Assert `data["role"] == "OWNER"`
- Assert `"slug" in data` and `"id" in data`
- Register: `teardown_registry["org_ids"].append({"slug": data["slug"]})`

**TORG-02: test_list_orgs_for_member(auth_client, org)**
- `org` fixture ensures at least one org exists
- GET `/organizations/`
- assert_status(resp, 200)
- Assert response is a list with `len(data) >= 1`
- Assert each item has `"role"` field
- Assert at least one org has `role == "OWNER"`

**TORG-03: test_update_org_as_owner(auth_client, org)**
- PATCH `/organizations/{org['slug']}/` with `{"description": "Updated by test"}`
- assert_status(resp, 200, "PATCH /organizations/{slug}/")
- Assert `data["description"] == "Updated by test"`
- CRITICAL: URL uses SLUG not UUID -- `/organizations/{org['slug']}/`

**TORG-04: test_org_slug_auto_generated_with_dedup(auth_client, teardown_registry)**
- Create two orgs with the SAME name: `name = org_name()`
- POST `/organizations/` with `{"name": name}` -- first org
- POST `/organizations/` with `{"name": name}` -- second org (same name)
- assert_status both 201
- `slug1 = resp1.json()["slug"]`, `slug2 = resp2.json()["slug"]`
- Assert `slug1 != slug2`
- Assert slug2 starts with slug1 (dedup adds "-1" suffix)
- Register both in teardown

Imports: pytest, factories.common.org_name, helpers.api.assert_status

  ```json
    "postal_code": "90210",
    "capacity": 200
  }
  ```
- assert_status(resp, 201, "POST venues")
- Assert response has: `name`, `id`, `address`, `city`, `state`, `postal_code`, `capacity`

**TVENU-02: test_list_venues_for_org(auth_client, org, venue)**
- `venue` fixture ensures at least one venue exists
- GET `/organizations/{org['slug']}/venues/`
- assert_status(resp, 200)
- Assert response is a list with `len(data) >= 1`

**TVENU-03: test_update_venue(auth_client, org, venue)**
- PATCH `/organizations/{org['slug']}/venues/{venue['id']}/` with `{"capacity": 999}`
- assert_status(resp, 200, "PATCH venue")
- Assert `data["capacity"] == 999`
- Note: venue URL uses venue UUID (`venue['id']`), NOT slug (venues don't have slugs)
- Restore original capacity after: PATCH back with `{"capacity": venue["capacity"]}`
